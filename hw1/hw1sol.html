  1. What’s the difference between the following two commands?

    echo 'hello, world' > test1.txt

    and

    echo 'hello, world' >> test2.txt

Solution: The first command directs output from one command to a file, so ‘hello, world’ is copied into a new script. The second command appends output from one command to a file, so ‘hello, world’ could be added to a new or previously existing script. Since ‘hello, world’ is appended to an empty script, the output from the first and second command appear to be the same.
